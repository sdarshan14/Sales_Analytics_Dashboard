# Sales Analytics Dashboard

## Overview

This project analyzes sales data using Python and performs exploratory data analysis (EDA) to identify business trends, product performance, profit distribution, and regional sales patterns.

The objective of this project is to transform raw sales data into meaningful business insights through data cleaning, analysis, and visualization.

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset

Dataset: Sample Superstore Dataset

The dataset contains information related to:

- Orders
- Products
- Sales
- Profit
- Customers
- Regions
- Categories

## Visualizations

### Sales by Category

Shows total sales generated by each product category.

### Profit by Category

Displays profit contribution from each category.

### Top 10 Products by Sales

Identifies the highest-performing products based on revenue.

### Sales by Region

Visualizes regional sales distribution.

### Monthly Sales Trend

Tracks sales performance over time.
